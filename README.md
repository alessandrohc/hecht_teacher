# Hecht Teacher

A small native macOS app that uses the OpenAI API to help you practice and
improve your English. Two features:

1. **Main window** ‚Äî paste a context on top, write your English reply at the
   bottom, and the AI rewrites it more naturally and explains what changed.
2. **System-wide Service** ‚Äî select any text in any app, right-click ‚Üí Services
   ‚Üí *Teach me: translate & coach* (or use a keyboard shortcut). A small
   floating panel appears near the mouse with a Portuguese translation,
   vocabulary glosses and (when relevant) a more natural English version.

## Requirements

- macOS 13 or newer
- Xcode 15+ command line tools (Swift 5.9+) ‚Äî you already have Xcode 26.5
- An OpenAI API key ‚Äî get one at <https://platform.openai.com/api-keys>

## Build and install

```bash
# Build the .app bundle in ./build/HechtTeacher.app
make bundle

# Build, then copy to ~/Applications and tell macOS to scan the Services entry
make install
```

After `make install`, open **Hecht Teacher** once from `~/Applications` (or
double-click the icon in Finder). Open Settings (`‚åò,`), paste your OpenAI API
key, choose a model (default: `gpt-4o-mini`, cheapest), and click Save. The key
is stored in your macOS Keychain ‚Äî not in plain files.

## Using Feature 1 ‚Äî improve my answer

1. Paste the message you are responding to in the **Context** box.
2. Write your answer in English in the **Your English answer** box.
3. Click **Improve my answer** (or press `Return`).

You'll get an improved version, a short bulleted list of what changed and why,
and one or two alternative phrasings.

## Using Feature 2 ‚Äî translate & coach the selected text

1. In **any** app (Mail, Safari, Notes, Messages‚Ä¶), select a word, phrase or
   sentence.
2. Right-click ‚Üí **Services** ‚Üí **Teach me: translate & coach**.
3. A small floating panel appears with the translation, vocabulary glosses and
   (if useful) a better English version.

### Assigning a keyboard shortcut

By default the service requests `‚åÉ‚å•‚åòT` (Control-Option-Command-T). If macOS does
not pick it up, set one manually:

1. Open **System Settings** ‚Üí **Keyboard** ‚Üí **Keyboard Shortcuts** ‚Üí **Services**.
2. Find **Teach me: translate & coach** under *Text*.
3. Click *Add Shortcut* and press the combination you want (e.g. `‚åò‚áßT`).

If the Service does not appear in the Services menu, run:

```bash
make reregister
```

and reopen the target app (or log out and back in).

## Useful commands

```bash
make build       # swift build -c release
make icon        # regenerate Resources/AppIcon.icns from Tools/MakeIcon.swift
make bundle      # also assemble HechtTeacher.app (icon included)
make install     # also copy to ~/Applications and register the Service
make run         # bundle + open the app
make reregister  # force macOS to rescan Services and lsregister
make clean       # remove .build/, build/ and generated icon files
```

## Icon

The app icon is generated by `Tools/MakeIcon.swift` ‚Äî a single-file Swift script
that draws PNGs at every macOS-required size with Core Graphics, then
`iconutil` packs them into `Resources/AppIcon.icns`. Edit the colors or shapes
in that script and run `make icon` to regenerate. The committed `.icns` is the
default and is consumed by `make bundle`.

## Notes for someone learning Swift

- Layout is built programmatically with `NSStackView` and Auto Layout ‚Äî no
  Storyboards or SwiftUI. This keeps every line of UI code visible and editable
  in plain Swift files.
- The OpenAI call uses `async/await` over `URLSession` ‚Äî no external packages.
- The Services menu integration relies on a properly assembled `.app` bundle
  with an `NSServices` entry in `Info.plist`. Running the binary directly from
  `.build/release/HechtTeacher` will show the window but the Services entry
  will not register ‚Äî always test with `make install` (or at minimum
  `make bundle && open build/HechtTeacher.app`).
